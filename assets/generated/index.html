    fast-vollib generated icon assets¶

    This directory contains ready-to-use raster exports derived from fast-vollib-icon.svg.

    Transparent PNG¶

    • fast-vollib-icon-16.png: favicon / browser tab
    • fast-vollib-icon-32.png: small web icon
    • fast-vollib-icon-48.png: desktop shortcut / launcher
    • fast-vollib-icon-64.png: docs / website use
    • fast-vollib-icon-128.png: general app and docs use
    • fast-vollib-icon-180.png: iOS-touch style size
    • fast-vollib-icon-192.png: Android / PWA icon size
    • fast-vollib-icon-256.png: medium app store / docs asset
    • fast-vollib-icon-512.png: high-resolution app / web asset
    • fast-vollib-icon-1024.png: App Store / presentation asset
    • fast-vollib-icon-2048.png: large-format slides / posters

    White-background JPG¶

    • fast-vollib-icon-512.jpg
    • fast-vollib-icon-1024.jpg
    • fast-vollib-icon-2048.jpg
    • fast-vollib-icon-presentation.jpg

    These are useful for PowerPoint, Keynote, Word, and environments that do not handle SVG or transparency well.

    Other exports¶

    • fast-vollib-icon-presentation.png: 1200x1200 transparent presentation-ready asset
    • fast-vollib-favicon.ico: favicon bundle built from 16px and 32px variants

    Social assets¶

    Files under social:

    • fast-vollib-github-social-1280x640.png
    • fast-vollib-github-social-1280x640.jpg

    Use these for GitHub social preview, Open Graph cards, and repository share links.

    LinkedIn assets¶

    Files under linkedin:

    • fast-vollib-linkedin-post-1200x627.png
    • fast-vollib-linkedin-post-1200x627.jpg
    • fast-vollib-linkedin-square-1080.png
    • fast-vollib-linkedin-square-1080.jpg

    The 1200x627 asset is the default landscape post format. The 1080 square asset is useful for carousel or square feed layouts.

    App and store assets¶

    Files under app:

    • fast-vollib-app-icon-1024.png
    • fast-vollib-play-icon-512.png
    • fast-vollib-play-feature-1024x500.png
    • fast-vollib-play-feature-1024x500.jpg
    • fast-vollib-appstore-portrait-1242x2688.png
    • fast-vollib-appstore-portrait-1242x2688.jpg

    These cover a high-resolution icon, a Play Store style feature graphic, and a portrait marketing panel suitable for App Store screenshots or promo slides.

    Monochrome variants¶

    Files under monochrome:

    • fast-vollib-monochrome-dark-512.png
    • fast-vollib-monochrome-dark-1024.png
    • fast-vollib-monochrome-light-512.png
    • fast-vollib-monochrome-light-1024.png

    These are useful for dark-mode favicons, watermarking, one-color print, or places where the full gradient icon is too busy.
